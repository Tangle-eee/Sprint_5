# UI Автотесты для проекта "QA Desk"

## 📋 Описание

Автоматизированные тесты с использованием `pytest` и `selenium` для проверки ключевых пользовательских сценариев веб-приложения [qa-desk.stand.praktikum-services.ru](https://qa-desk.stand.praktikum-services.ru/).

## 🔍 Покрытые сценарии

- ✅ Регистрация пользователя (валидная, с невалидным email, уже существующий пользователь)
- ✅ Вход в систему (Login)
- ✅ Выход из системы (Logout)
- ✅ Создание объявления (авторизованный и неавторизованный пользователь)

## 📁 Структура проекта
.
├── conftest.py # Фикстура для инициализации драйвера
├── locators.py # Локаторы элементов UI
├── test_register.py # Тест на успешную регистрацию
├── test_register_invalid_email.py # Тест с невалидным email
├── test_register_existing_user.py # Тест с уже зарегистрированным пользователем
├── test_login_user.py # Тест логина
├── test_logout_user.py # Тест логаута
├── test_create_ad_authorized.py # Тест на создание объявления (авторизован)
├── test_create_ad_unauthorized.py # Тест на создание объявления (без авторизации)
